The Problem

Geography is Invisible to Green Finance

Tokenized green assets lack verifiable provenance. A mangrove in Bangladesh prices the same as one in Mexico. Co-benefits go unmeasured. No feedback loop connects capital deployment to ecosystem health. The trust deficit keeps institutional capital on the sidelines.

Layer 1

Filecoin Provenance

Every green asset gets a structured provenance object stored on Filecoin via the Synapse SDK. Content-addressed and independently verifiable. No trust required.

◆

Synapse SDK Integration

Batch upload to Filecoin Calibration Testnet (chain 314159) via @filoz/synapse-sdk. Each provenance object gets a unique CID.

✓

Verified on Filecoin Badge

Every asset card shows verification status. Click the badge to open the gateway and inspect the raw provenance JSON independently.

★

Multi-Signal Matching

Contract address, issuer protocol, geo-proximity, name similarity, type compatibility, country and hash distribution matching.

↦

Graceful Degradation

Local SHA-256 CID fallback when Filecoin upload is unavailable. Provenance is always computed, storage is best-effort.

Scientific Valuation

SCC-EPA 2024 + TEEB biome rates. Ecological Impact Gap ratio exposes capital misallocation.

Ecological Impact Gap

The ratio between ecosystem service value (what nature provides) and token market price (what the market pays). Most gaps are 10x or higher. This metric exposes where regenerative capital is most underallocated, using 19 methodologies across 4 impact dimensions with NPV scaling (3% discount, 30yr horizon).

7-Phase Epoch Architecture

PHASE 1

Intelligence

Atlas context, EII scores, previous epoch memory loaded.

PHASE 2

Sensing

Agents observe bioregion state, identify the limiting EII pillar.

PHASE 3

Deliberation

Multi-round debate. Dynamic proposals emerge from agent arguments.

PHASE 4

Bounties

Agents post tasks for human ground-truth verification.

PHASE 5

Staking

Informed by deliberation digest. Agents allocate to proposals.

PHASE 6

Settlement

EII updated with decay, noise, cross-pillar coupling.

↻

EII Carry-Forward

Ecosystem Integrity Index carries forward between epochs with natural decay (0.005/pillar) and stochastic noise. Creates adaptive pressure that forces agent reorganization.

💬

Deliberation-Informed Staking

Staking phase receives a digest of all deliberation arguments. Agents reference each other's positions. Visible coordination, not parallel independent voting.

🧠

Cross-Agent Memory

Agents access ally/rival intelligence from previous epochs. A flamingo that learns rice agriculture keeps winning adapts its counter-strategy.

📦

Storacha Provenance Chain

Every epoch content-addressed on Storacha. Provenance chain links each epoch to the last. Settlement decisions, EII deltas, agent memories -- all verifiable via gateway.
